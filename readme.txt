== Description ==

Remove outdated revisions of posts in one click.

== Installation ==

1. Upload `plugin-name.php`and includes directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= All revisions will be removed simultaneously in one click? =

Yes. All revisions are removed simultaneously in a single click.
